# Troubleshooting

## Installer cannot verify the repository

Stop installation and confirm that the source repository is:

```text
https://github.com/laudoluxDev/realtortasksai-mcp
```

## Browser login times out

Run the installer again, or use the license-key fallback when prompted.

## Tools do not appear

Restart the MCP client after installation. Claude Desktop, Cursor, Windsurf, and
Codex usually load MCP configuration on startup.

## Node.js or npm is missing

`npm` comes with Node.js. Install the current LTS version from:

```text
https://nodejs.org/
```

After Node.js is installed, rerun the official GitHub install prompt.
